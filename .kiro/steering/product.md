# Product Summary

This is an **Agentic AI Training** project — a learning workspace for building agentic AI applications using LangChain, LangGraph, and related tooling.

The project focuses on:
- Building AI agents with LangChain and LangGraph
- Retrieval-Augmented Generation (RAG) with vector stores (FAISS, ChromaDB)
- Graph-based knowledge retrieval with Neo4j
- Model Context Protocol (MCP) integration
- Observability via LangSmith
- Multi-agent orchestration patterns (CrewAI exploration)

This is primarily an educational/experimental workspace with Jupyter notebooks and Python scripts.
